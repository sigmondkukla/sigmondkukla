CUR Payload Datalogger

To support our Payload subteam's goal of characterizing the performance of various insulation options for cryogenic cargo, I planned, designed, and produced a data logging system that interfaces with four thermocouples integrated within the payload, recording this data throughout the rocket's integration, preparation for launch, flight, and recovery. The data collected informs future decisions the team may make when designing a hybrid or liquid rocket that requires cryogenic propellant. As the payload is not readily accessible from outside the rocket, an ESP32-C3 microcontroller with wireless capability was used to allow for remote control via a web interface.

In developing this system, I have refined my skills in:

  • Custom PCB design
  • Firmware development for embedded systems
  • Efficiently storing sensor data to a flash memory
  • Creating a lightweight, mobile-friendly web interface for remote control and data visualization
